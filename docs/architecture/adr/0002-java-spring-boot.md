# ADR-0002: Java 21 ve Spring Boot 3.x

## Karar

Backend servisleri Java 21 ve Spring Boot 3.x ile gelistirilecek.

## Neden

Dokuman Java ve Spring Boot bekliyor. Java 21 guncel LTS tabani saglar.
Spring Boot 3.x; Spring Security, Spring Data JPA, observability ve modern
container tabanli uygulamalar icin guclu ekosistem sunar.

## Sonuc

Kurumsal standartlara yakin, uzun vadede bakimi daha kolay bir backend zemini
olusur.
