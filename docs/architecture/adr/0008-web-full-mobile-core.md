# ADR-0008: React Web Tam Kapsam, React Native Mobil Temel Kapsam

## Karar

React web istemcisi ana operasyon deneyimini tam kapsayacak. React Native mobil
istemci login, ticket liste, detay, ticket acma ve temel notification akisini
kapsayacak.

## Neden

Web ve mobilin esit kapsamda gelistirilmesi teslim maliyetini gereksiz artirir.
Sadece web yapmak ise dokumandaki mobil beklentisini zayif birakir.

## Sonuc

Ana is akislari web'de guclu olur. Mobil temel isteri karsilar ve ileride
genisletilebilir.
