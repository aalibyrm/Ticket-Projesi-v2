# Apps

Client uygulamalari bu dizinde tutulur.

- `web`: React 18, TypeScript, Material UI v7, React Router Framework, Redux
  Toolkit, TanStack Query ve OpenAPI generated REST client ile gelistirilen web
  istemcisi. Ana operasyon deneyimi burada gelistirilir.
- `mobile`: Expo Managed + React Native + TypeScript mobil istemcisi. Temel
  ticket akislari burada gelistirilir.
